How to use WiFOut:

    • 1. Login to the web application by using the network admin user credentials
      2. On the dashboard page, the scanned devices are displayed.
      3. The user can block a device by clicking the block button.
      4. The user can allow a device by clicking the allow button.
      5. On the blocked devices tab, a device can be removed from the blocked database by clicking the remove button.
      6. On the allowed devices tab, a device can be removed from the allowed database by clicking the remove button.
      7. Once a device was added to the blocked database the system will request the wireless network adapters to send
      deauthentication packets to rogue access points.

    • For more details on how to use the system, please download this PDF Manual: User Manual for WiFOut

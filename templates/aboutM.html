About WifOut

WiFOut is a wifi immobilizer which used the capabilities of script developed with python programming language. The script was integrated and managed by a web application resulting to a system. The system has the potential to remove the use of rogue access points in the company by sending deauthentication packets with the use of wireless network adapters.
